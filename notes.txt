Use Arduino Uno, genuine w/ Atmega16U!

+ Switch easily between Arduino programming and gamepad mode
- Only 8-bit stick inputs (instead of 10-bit available) due to communication between Atmega32 and Atmega16
- Makefile needed to be fixed to run
  MCU = atmega8u2
  F_CPU = 8000000
  
Compiling of the Atmega16/Atmega8:
$ sudo apt-get install dfu-programmer gcc-avr avr-libc

Change the USB Vendor/Product ID (VID:PID) in usb_gamepad.c
https://www.the-sz.com/products/usbid/index.php?v=0x046D
VID:PID: 10c4:82c0
=> 0x046D: 0xC260 voor Logitech G29 (PS4)
